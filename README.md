# DRIFFFT (v2) HTML/CSS/JS + GSAP

## Run locally
Open `index.html` in your browser.

Recommended:
- VS Code Live Server
- Or: `python3 -m http.server 8080` then open `http://localhost:8080`

## Add your product links
In `index.html`, replace the two `href="#"` values on the product cards with your real URLs.
